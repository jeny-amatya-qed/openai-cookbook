# Customer
## Jobs
- View requirements
- Connect to APIs
- Manage my account
- Test my product
- Check compatibility

## Pain points
- Vague standards
- Unavailable services
- Slow responses
- Lack of tools
- Moving goal posts

# Value
## Services
- Standard & practices
- Style guides
- API visibility

## Pain relief
- Clear, concise, timely and accurate information
- Open and responsive tools
- Rapid feedback and support tools
- Live support
